# 11. PRODUCTION SOP: IDEA TO SHIP

## Six Non-Negotiables
1. **Rights and provenance by design** — asset register with source, license, model/tool, prompt, date, contributor
2. **Human authorship explicit** — separate "AI-assisted drafting" from "human-authored final"
3. **Single source of truth** — one canon library for lore, timeline, characters, UI rules
4. **Continuity engineering** — scene IDs, beat sheets, dependency tracking, contradiction checks
5. **Prompt and style governance** — prompts as production assets with versioning
6. **Build-measure-learn loops** — fast iteration with structured testing gates

## Five Production Lanes (Parallel, Gated)
1. Narrative (treatment → script → branching logic)
2. Visual (storyboards → key art → backgrounds → UI)
3. Character (design → expression sets → voice → animation)
4. Audio (VO, SFX, music, mixing, localization prep)
5. Implementation (engine integration, QA, release ops)

## Step-by-Step Process
- **Step 0 (1–3 days):** Production setup — bible, asset register, naming conventions, quality bars, AI standards
- **Step 1 (2–5 days):** Concept lock — logline, themes, ICP profile, core loop
- **Step 2 (5–10 days):** Treatment and beat sheet — structure, scene IDs, branch map, continuity check
- **Step 3 (2–4 weeks):** Scripting — engine-ready format, choice design, table read package
- **Step 4 (1–3 weeks):** Character development — visual/narrative/performance bibles, expression sets
- **Step 5 (1–2 weeks):** Storyboard and shot planning — scene boards, key frames, UI overlays
- **Step 6 (2–8 weeks):** Art production — BGs, sprites, CGs, UI kit
- **Step 7 (1–4 weeks):** Audio production — VO plan, SFX library, music cues
- **Step 8 (2–6 weeks):** Integration and build — engine implementation, Steam page assets
- **Step 9 (2–4 weeks):** QA, playtests, polish — bug triage, pacing fixes, final polish

## Episode 1 Schedule Template (~11 weeks)
- Week 1: Setup + concept lock
- Week 2: Treatment + beat sheet + character bibles v1
- Weeks 3–4: Script v1 → table read → lock; storyboards + key frames
- Weeks 5–7: Art production + audio production
- Weeks 6–8: Integration + first playable
- Weeks 9–10: QA, playtests, polish
- Week 11: Release packaging + store assets + compliance

## Production Gates
- **Gate A (Treatment Lock):** Stakes, desire, theme, end promise clear
- **Gate B (Script Lock):** Every scene has purpose, choices meaningful
- **Gate C (Visual Lock):** Character consistency stable, UI readability proven
- **Gate D (Playable Lock):** No dead ends, no missing assets, pacing holds
- **Gate E (Release Lock):** Platform disclosure, rights documentation, asset register complete
