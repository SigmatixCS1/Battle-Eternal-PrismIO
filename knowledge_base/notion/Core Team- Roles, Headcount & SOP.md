# Core Team: Roles, Headcount & SOP

For a 90‑minute, fully animated, commercial‑quality VN episode using AI‑accelerated tools, you can ship with a tight core team of 6–8 people plus 3–5 part‑time/outsourced specialists, if you run a disciplined pipeline. Visual novels often combine roles compared to larger studios, but the key functions below still need clear owners.

Below is a concrete SOP from concept to ship, with roles, core responsibilities, and where AI gives you the 10x. Assume ~4–6 months calendar time with overlapping phases, not strictly sequential.

Minimum always‑on team (6–8 people)

Creative Director / Showrunner

Owns: Vision, tone, mythology, final creative decisions.

Responsibilities:

Lock series bible, episode theme, and theological spine.

Approve characters, boards, script, audio, and marketing beats.

Make cut decisions when scope threatens schedule/budget.

Background: Senior writer or director with VN/anime and myth/lore literacy.

Narrative Designer / Lead Writer

Owns: Story architecture, branching logic, script.

Responsibilities:

Turn the vision into treatment, beat sheets, and route flowcharts.

Write or supervise all dialogue, choices, and conditional content.

Ensure every choice has visible impact, avoid fake branching.

Background: VN or game narrative design; comfortable with interactive structure.

Art Director / Lead Artist

Owns: Visual language, character and world consistency.

Responsibilities:

Define style guide (line, color, lighting, animation level).

Supervise character, CG, background, and UI art.

Use AI concepting responsibly; ensure final art meets “prestige VN” bar.

Background: 2D character/background artist with art‑lead experience.

Technical Director / VN Engineer (Ren’Py or engine of choice)

Owns: Pipeline, builds, scripting, tools.

Responsibilities:

Implement scenes, branching, flags, save/load, and platform‑specific requirements.

Build AI‑assisted tools (script importers, asset auto‑wiring, batch exporters).

Ensure performance, stability, and easy debug for writers and artists.

Background: Game programmer with VN/engine experience.

Producer / Project Manager

Owns: Schedule, budget, staffing, prioritization.

Responsibilities:

Define milestones (concept, script lock, content complete, gold).

Coordinate external contractors, QA, and localization.

Keep burndown, risks, and scope managed.

Background: Game or animation production; familiar with agile/lightweight tools.

Lead 2D Generalist (Characters + CGs)

Owns: Majority of character sprites, expressions, and CG event art.

Responsibilities:

Produce final/painted versions of AI‑assisted concepts and key visuals.

Work with animation tools (Live2D, Spine, etc.) or hand‑animate simple loops.

Background: VN or anime‑style character artist, comfortable with rig‑friendly art.

Background / UI Artist (can be 0.5–1 FTE)

Owns: Environments, UI skin, icons.

Responsibilities:

Generate backgrounds via AI + paint‑over, or from scratch where needed.

Create VN UI (text box, menus, choice widgets) and FX overlays.

Background: Environment art & graphic design.

Audio Lead (Composer / Sound Designer) (0.5 FTE)

Owns: BGM palette, key themes, SFX strategy.

Responsibilities:

Compose adaptable tracks for mood shifts and route variants.

Build reusable SFX library and basic audio mix.

Background: Game or VN music; can be freelance with recurring engagement.

Specialists (part‑time / contract as needed)

Editor / Script Supervisor (narrative clarity, localization‑friendly text)

QA Lead (test plans, regression, console compliance if needed)

Marketing / Community Manager (store page, trailers, Discord, VTuber outreach)

Localization partners (if shipping multilingual)

Legal/Business (contracts, IP, platform agreements)

### SOP: Concept to Shippable 90‑min Episode

Treat this as a checklist per episode. The same skeleton repeats each ep once the season’s spine is set.

### Phase 1 – Concept & Greenlight (2–3 weeks)

1.1 Vision & constraints

Creative Director + Producer + Narrative Designer

Confirm: episode length (≈90 minutes main route), target ship window, platforms.

Lock high‑level premise for the episode (theological conflict, character focus, key twist).

1.2 Mythology & character focus

Narrative Designer + Creative Director

Identify which myth arcs and which character arcs move this episode.

Decide mandatory story beats that cannot be branched away from (spine).

1.3 AI‑assisted concept exploration

Narrative Designer, Art Director

Use LLMs to brainstorm scenarios, myth inversions, and setpieces.

Use image models for moodboards (locations, lighting, symbolic motifs).

Human leads prune; nothing goes forward without explicit acceptance.

Deliverables:

3–5 page episode treatment.

Updated season arc map showing this episode’s contribution.

Moodboard for visual tone and key environments.

### Phase 2 – Story Architecture & Script (4–6 weeks)

2.1 Route & choice design

Narrative Designer + Creative Director + Technical Director

Build a simple route flowchart:

Core path everyone sees.

Optional scenes and at least 2–3 meaningful forks with visible consequences.

Technical Director notes implementation complexity and flags scope issues.

2.2 Beat outline & pacing

Narrative Designer

Break 90 minutes into beats (intro, rising tension, midpoint, crisis, resolution).

Tag beats with: relationship, mythology exposition, horror spike, choice moment.

2.3 AI‑drafted scenes

Narrative Designer (with LLM)

Generate first‑pass dialogue and alternative reactions within each beat.

Hard rules: consistent character voice, no marketing‑speak, no fake emotional swings.

2.4 Human rewrites & edit

Narrative Designer + Editor

Restructure for clarity, subtext, and route coherence.

Reduce over‑branching; aim for replay value without exploding content.

2.5 Script‑to‑engine prep

Technical Director + Writer

Standardize script format (labels, speaker tags, branch markers) for automated import.

Deliverables:

Locked episode script (main and branch content).

Route flowchart annotated with estimated line counts and scene durations.

### Phase 3 – Visual Design & Storyboards (3–4 weeks, overlaps Phase 2)

3.1 Character and key art

Art Director + Lead 2D Generalist

For any new or updated character looks:

AI concepting for silhouettes/poses; human clean‑up and style enforcement.

Define expression sets and costume variants needed for this episode.

3.2 Environment & UI needs

Background/UI Artist + Art Director

List all locations for the episode.

Decide re‑use vs new creation; generate AI base for new BGs then paint‑over.

3.3 Storyboard / animatic

Art Director + Creative Director

Board critical scenes: openings, choice reveals, climax CGs.

Use AI to generate rough panel ideas, but finalize boards manually for clarity and framing.

Deliverables:

Final character sheets and expression lists for this episode.

Approved backgrounds list and UI updates.

Compact animatic or still board set for ~10–20 key scenes.

### Phase 4 – Asset Production (6–8 weeks, overlaps Phase 3–5)

4.1 Character art & animation

Lead 2D Generalist + Art Director

Produce final sprites, expressions, and 3–6 key CGs.

Use AI line‑assist/inbetween tools for animations; manual polish on faces and hands.

4.2 Backgrounds & FX

Background/UI Artist

Deliver final backgrounds at required resolutions.

Create simple FX overlays (glitches, lighting, divine manifestations).

4.3 UI & system visuals

Background/UI Artist + Technical Director

Implement any episode‑specific UI (new choice styles, flashback frames).

4.4 Audio production

Audio Lead

Compose and mix new tracks or variations needed for this episode.

Collect SFX; use AI tools for sweetening and stem management where helpful.

Deliverables:

All 2D art and animation for the episode.

Audio assets (BGM, SFX, stingers) organized by cue sheets.

### Phase 5 – Implementation & Integration (3–5 weeks)

5.1 Script import & scene wiring

Technical Director

Import script into engine, wire dialogue, choices, conditions, and flags.

Implement auto‑skip, backlog, saves, and accessibility settings.

5.2 Timing & camera polish

Technical Director + Art Director + Writer

Adjust pauses, camera moves, and transitions to enhance emotional beats.

5.3 Audio integration

Technical Director + Audio Lead

Trigger music changes on emotional shifts and branch transitions.

Balance overall volume and SFX against BGM.

Deliverables:

Fully playable episode from title to credits on target platform.

### Phase 6 – QA, Tuning, and Compliance (3–4 weeks)

6.1 Functional QA

QA Lead + Internal testers

Test all routes for crashes, softlocks, and logic bugs.

Check choice flags and endings; verify no “dead” variables.

6.2 Narrative & pacing QA

QA + Writer + Director

Watch for pacing drag, confusing mythology moments, or tonal shifts.

Adjust text speed, transitions, and minor line edits.

6.3 Localization & build checks

Localization partners + Technical Director

Run placeholder machine translations through pipeline, then human‑edit.

Validate fonts, text expansion, and line breaks.

Deliverables:

Release candidate build, with known minor issues triaged or deferred.

### Phase 7 – Launch Readiness & Shipping (2–3 weeks)

7.1 Store assets & trailer

Creative Director + Producer + Marketing / Community Manager

Cut a trailer using in‑engine footage.

Prepare screenshots, key art, and store copy.

7.2 Platform submission

Technical Director + Producer

Package builds, upload to Steam/itch, complete checklists.

7.3 Launch plan execution

Marketing / Community + Producer

Coordinate VTuber codes, embargoes, and social beats.

Set up announcement and patch cadence.

Deliverables:

Episode live on storefronts; marketing beats in motion.

### Putting It Together: Team Size vs Scope

For a single 90‑minute episode at commercial quality with heavy AI assistance:

Minimum sustainable core: 6

Creative Director

Narrative Designer / Writer

Art Director (also main character artist)

Technical Director / Programmer

Producer

Audio Lead (part‑time)

Recommended to keep quality and speed: 8–10

Add Background/UI specialist

Add QA + Editor (shared across episodes)

Add Marketing / Community Manager (0.5 FTE)

This structure mirrors common VN team role lists (writer, character artist, background artist, GUI designer, programmer, editor, composer) while consolidating and layering AI efficiency into each responsibility instead of replacing it.
