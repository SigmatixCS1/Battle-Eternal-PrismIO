# Battle Eternal PrismIO
Creative development vault for the **Battle Eternal** universe — a dystopian narrative spanning visual novels, TCG, and transmedia formats.
## Overview
Battle Eternal is set in a world governed by **The Hegemony**, a one-world government that controls society through the **F-Link** system. The story follows a cast of characters navigating metaphysical awakenings, secret orders, and ideological warfare across multiple interconnected arcs.
This repository serves as the **narrative design and character architecture hub**, housing scene scripts, character profiles, structural documents, and the AI-assisted Character Architect system.

## Foundational Infrastructure
### Corporate Power Hierarchy
The Battle Eternal world operates through a layered system of corporate, institutional, and occult power:
1. **Surface Civilization** — The Hegemony (visible world)
2. **Functionality Link Systems (F-Link)** — Consumer-facing neural technology corporation (HQ: Neo-Osaka). Operates the F-Link Neural Interface and the hidden F-Core adaptive intelligence kernel. Divisions feed from Saint Radian houses.
3. **Sable Dynamics** — Hidden parent conglomerate controlling global infrastructure (energy, water, agriculture, logistics). Subsidiaries: Sable Neurotech, Sable BioMetrics, Cognitive Wellness Partners, Sable Defence Systems.
4. **The Meridian Group** — Multi-sector holding company owning Sable Dynamics, Prisma Media Networks, NutriCore Distribution, Aethon Transit Systems, and the Saint Radian Educational Trust.
5. **The Concordance** — Intergenerational shadow governance council (13 seats, founded 17th century by Isabella de la Fonte).
6. **Order of the Black Sun** — Ancient occult command network. Leader: Cassian Thorne.
7. **Server Zero / Dark Hold** — Metaphysical engine of extraction at the base of the entire system.
Information flows upward; control flows downward. Each layer believes it is the top.
### Saint Radian Houses (8 Houses)
Saint Radian Academy organizes students into eight Houses, each representing a fundamentally different approach to controlling reality:
- 🜂 **Order of the Obsidian Glyphs** — Meaning → Inscription (Dorm: The Sepulchre, Apex: Aurelian Voss)
- ⚙️ **The Aetherion Society** — Systems → Reconstruction (Dorm: Titan's Workshop, Apex: Caelum Virex)
- 🌙 **The Astral Chorus** — Emotion → Resonance (Dorm: The Lyceum, Apex: The Four Sopranos)
- 🔥 **House Pyraxis** — Will → Combustion (Dorm: The Crucible, Apex: Rhazek Valen)
- 🕶️ **House of the Liminal Veil** — Perception → Distortion (Dorm: Nadir Hall, Apex: Veyra Null)
- 🌿 **House of Living Sigils** — Life → Evolution (Dorm: The Verdant Spire, Apex: Eidren Solvale)
- ♾️ **House Aeonis** — Radiance → Projection (Dorm: The Empyrean Spire)
- 🏛️ **House Dialectica** — Truth → Deconstruction (Dorm: The Agora)
### Key Corporate Systems
- **F-Core** — Embedded intelligence within F-Link with four protocol layers (Stability, Optimization, Observation, Emergent)
- **Sin Eater System (SES)** — Classified F-Link extension for redistributing psychological burden; creates "Hollows" through a 5-phase process
- **Blackout Protocol** — Sable Dynamics' 4-phase neutralization system (Acquisition → Regulatory Suffocation → Economic Collapse → Terminal Resolution)
- **The Hidden War** — Covert ideological conflict between Sable Dynamics (infrastructure control) and the Order of the Black Sun (mythological control)

## Core Lore Systems
- **Inscription Arcana (Ars Inscripta)** — hidden occult-linguistic protocol that converts human emotion, dream residue, and narrative behavior into living runic code via F-Link.
- **Terra Trion Gate (Sigilum Trionis)** — planet-scale ritual geometry formed by three Saint Radian academies (Japan, Greece, Kenya) functioning as Memory, Language, and Emotion anchors.
- **Pantheon Convergence Event** — convergence point where archetypal density and Dreamfield compression ignite the Pantheon War.
- **Dreamfield Cosmology** — layered metaphysical architecture (Null Author → Cosmic Lattice → Omega Entities → Dreamfield → Pantheon/Primal systems → civilization/technology).
- **Five Omega Entities** — Sigmatix, Nemesis, Logos, Dream Sovereign, and Null Author as equilibrium forces above host-level mythic systems.
- **Mantle Taxonomy** — mantle classes (Solar, Judgment, War, Trickster, Underworld, Nature, Sovereignty) tracked via Order classification frameworks.
- **Pantheon Awakening Ladder** — progressive 6-stage host emergence model from Dormant Host to Apotheosis.
- **Saint Radian House System** — eight domain-based Houses (Meaning, Systems, Emotion, Will, Perception, Life, Radiance, Truth) with apex leadership figures, ability tier systems, and a Division Engine–fed Sorting Ceremony.
- **F-Link / F-Core Architecture** — neural interface system with hidden adaptive intelligence governing perception, behavior, and anomaly detection across the global population.
- **Corporate Hierarchy (Isabella de la Fonte's Design)** — 400-year layered camouflage system: FLS → Sable Dynamics → Meridian Group → Concordance → Order of the Black Sun → Dark Hold.
- **Spiral Codex** — oldest surviving prophetic framework linking alchemical stages (Nigredo, Albedo, Citrinitas, Rubedo) to the four-stage cycle of civilizational rise and collapse.
- **Nemesis Cycle / Nemesis Pattern** — cosmological correction mechanism that activates when technological hubris, power concentration, emotional exploitation, and mythic saturation cross critical thresholds simultaneously.
- **Black Sun Reality Engine** — the Order's planetary-scale infrastructure designed to intercept and override the Nemesis Pattern.
- **Aetherforge Program** — classified Aetherion Society initiative to reduce spellcraft to code via SpellVectors, threatening the power bases of rival Houses.
- **Sin Eater System** — classified burden redistribution technology creating an invisible underclass of Hollows through F-Link neural manipulation.
